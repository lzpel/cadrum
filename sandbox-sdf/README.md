## おもっていること1

SDFから元の曲線を再現するには

ある点の近傍にある境界sdf=0が直線なのか曲線なのか知る必要がある

で曲線は円弧かbsplineしか無いと考えていいがbsplineは現実的には無視して良い気がする。つまり曲線なら円弧だとみる。

そうするとエンティティは直線か円弧の2つだけで、その二つで閉領域が記述できると仮定する

そうすると曲線か直線かを判別する必要がある

### アイディア

でアイディアとしてlib.rsの
fn nabla(p: Vec2, sdf: impl Fn(Vec2) -> f32) -> Vec2
を
fn nabla_laplacian(p: Vec2, sdf)->Vec2, Float
として曲率=0なら直線　曲率!=0なら曲線とする案
ちょうどp+前後左右4点の中心差分をつかっているから十分な情報があるはず
